# MoodWorld 上線教學（新手版，照著點就好）

準備一個 Email 就能開始（Supabase / GitHub / Vercel 三個都可以用 Google 帳號登入，比較快）。

---

## 步驟一：Supabase（資料庫）

1. 瀏覽器開 **supabase.com**，右上角點 **Start your project**，用 Google 或 GitHub 帳號登入。
2. 登入後點 **New Project**。
3. 填資料：
   - Name：`moodworld`（隨便取都可以）
   - Database Password：自己設一個密碼，**先存到記事本**，等一下會用到
   - Region：選離你比較近的（例如 Southeast Asia (Singapore)）
4. 點 **Create new project**，等 1–2 分鐘讓它建好。
5. 建好後，左側選單點 **SQL Editor**（`</>` 圖示）。
6. 點 **New query**。
7. 打開你電腦上 `moodworld/db/schema.sql`（用記事本開就好），全選複製，貼進 SQL Editor。
8. 點右下角綠色 **Run**，看到「Success. No rows returned」就是成功了。
9. 再點一次 **New query**，打開 `moodworld/db/seed.sql`，一樣複製貼上、點 **Run**。
10. 左側選單點 **Project Settings**（齒輪圖示）→ **Database**。
11. 往下找到 **Connection string**，選 **Connection pooling**（注意 port 是 `6543` 那一組，不是預設的 5432）。
12. 複製那串網址，把裡面的 `[YOUR-PASSWORD]` 換成步驟 3 你設的密碼，整串存起來——這串就是等下要用的 `DATABASE_URL`。

---

## 步驟二：把程式碼放到 GitHub（用 GitHub Desktop，不用打指令）

1. 到 **desktop.github.com** 下載安裝 **GitHub Desktop**。
2. 打開後用 GitHub 帳號登入（還沒有帳號的話，先到 github.com 免費註冊一個）。
3. 左上角 **File → Add local repository**。
4. 選你的專案資料夾：`C:\Users\qqkev\Claude\Projects\這世界的每個人的心情\moodworld`
5. 如果跳出「這不是 git repository，要建立一個嗎」，選 **create a repository**。
6. 左下角隨便填一句 commit 訊息（例如 `first version`），點 **Commit to main**。
7. 右上角點 **Publish repository**，Name 填 `moodworld`，點 **Publish**。

---

## 步驟三：Vercel 部署

1. 到 **vercel.com**，右上角 **Sign Up** → 選 **Continue with GitHub**（用同一個 GitHub 帳號）。
2. 登入後點 **Add New... → Project**。
3. 列表裡找到 `moodworld`，點 **Import**。
4. Framework Preset 會自動顯示 **Next.js**，不用改。
5. 往下找 **Environment Variables**，新增一筆：
   - Key：`DATABASE_URL`
   - Value：貼上步驟一存的那串連線字串
6. 點 **Deploy**，等 1–3 分鐘。
7. 完成後會出現一個網址，像 `moodworld-xxxx.vercel.app`——這就是你的正式網站了。

---

## 驗證有沒有成功

- 打開網址，投一次今天的心情。
- 換手機或無痕視窗再開一次網址，切到「Global」分頁，確認剛剛那一票有算進去。

## 之後要更新怎麼辦？

以後程式碼有改動，只要在 **GitHub Desktop** 裡點 **Commit** → **Push origin**，Vercel 會自動偵測並重新部署，不用重做上面的步驟。
